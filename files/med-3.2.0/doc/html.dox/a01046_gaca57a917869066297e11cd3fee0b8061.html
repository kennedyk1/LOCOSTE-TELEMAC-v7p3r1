med_err MEDlibraryStrVersion ( char *const   medversion  ) 

Renvoie le numéro de version de la librairie MED dans une chaîne de caractères.

Paramètres:
medversion Numéro de version défini dans une chaîne de caractères.
Valeurs retournées:
med_err retour négatif en cas d'erreur, Zéro sinon.

Cette routine retourne le numéro de version de la bibliothèque MED. Le numéro de version est renvoyé sous la forme d'une chaîne de caractères.
