subroutine mfdnfc ( integer  fid,
integer  ind,
integer  n,
integer  cret 
)

Cette fonction lit le nombre de composantes d'un champ.

Paramètres:
fid Identificateur du fichier.
ind Itérateur. L'itérateur commence à 1.
n Nombre de composantes.
cret retour négatif en cas d'erreur, Zéro sinon.
Voir également:
MEDfieldnComponent

Cette fonction lit le nombre de composantes d'un champ. L'indice correspond à l'indice du champ dans le fichier.
