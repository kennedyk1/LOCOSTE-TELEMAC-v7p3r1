med_err MEDparameterValueWr ( const med_idt  fid,
const char *const   paramname,
const med_int  numdt,
const med_int  numit,
const med_float  dt,
const unsigned char *const   value 
)

Cette routine permet l'écriture de la valeur d'un paramètre numérique scalaire.

Paramètres:
fid Identificateur du fichier.
paramname Nom du paramètre de longueur maximum MED_NAME_SIZE .
numdt Numéro de pas de temps de l'étape de calcul (MED_NO_DT si pas de numéro de pas de temps).
numit Numéro d'itération de l'étape de calcul (MED_NO_IT si pas de numéro d'itération).
dt Date du pas de temps si le numéro de pas de temps est différent de MED_NO_DT.
value Tableau des valeurs.
Valeurs retournées:
med_err retour négatif en cas d'erreur, Zéro sinon.

Cette routine permet l'écriture de la valeur d'un paramètre numérique scalaire pour une séquence de calcul donnée.
