subroutine msmnan ( integer  fid,
character *(*)  name,
integer  naxis,
integer  cret 
)

Cette routine permet de lire dans un maillage support le nombre d'axes du repère des coordonnées des noeuds avec accès direct.

Paramètres:
fid Identificateur du fichier.
name Nom du maillage, de longueur maximum MED_NAME_SIZE .
naxis Nombre d'axe dans le repère de coordonnées.
cret retour négatif en cas d'erreur, Zéro sinon.
Voir également:
MEDsupportMeshnAxisByName

Cette routine permet de lire dans un maillage support le nombre d'axes du repère des coordonnées des noeuds. le nombre d'axe correspond à la dimension de l'espace de calcul. L'accès au maillage support se fait directement via son nom.
