subroutine mfdcre ( integer  fid,
character*(*)  fname,
integer  ftype,
integer  ncomp,
character*(*)  cname,
character*(*)  cunit,
character*(*)  dtunit,
character*(*)  mname,
integer  cret 
)

Cette fonction crée un champ dans un fichier.

Paramètres:
fid Identificateur du fichier.
fname Nom du champ, de longueur maximum MED_NAME_SIZE .
ftype Type numérique des composantes du champ.
ncomp Nombre de composantes.
cname Nom des composantes du champ. Les noms des composantes sont définis dans une chaîne de nbcomponent * MED_SNAME_SIZE caractères.
cunit Unité des composantes du champ. Les noms des unités des composantes sont définis dans une chaîne de nbcomponent * MED_SNAME_SIZE caractères.
dtunit Unité des dates des séquences de calcul du champ. Elle est définie dans une chaîne de taille MED_SNAME_SIZE .
mname Nom du maillage, de longueur maximum MED_NAME_SIZE .
cret retour négatif en cas d'erreur, Zéro sinon.
Voir également:
MEDfieldCr

Cette fonction crée un champ dans un fichier. Un champ est composé d'une ou plusieurs composantes scalaires. A chaque composante est associé un nom et une unité. Le type des valeurs des composantes peut être au choix (med_field_type) :

  • MED_FLOAT64 : flottant 64 bits,
  • MED_INT32 : entier 32 bits,
  • MED_INT64 : entier 64 bits.

Définition à la ligne 20 du fichier medfield.f.
