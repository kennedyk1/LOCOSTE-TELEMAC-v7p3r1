subroutine mlbclo ( integer  cret  ) 

Cette routine force l'écriture des données sur disque, nettoie la mémoire et ferme tous les fichiers MED ouverts (de même que tous les fichiers HDF).

Paramètres:
cret retour négatif en cas d'erreur, Zéro sinon.
Voir également:
MEDlibraryClose

Cette routine force l'écriture des données sur disque, nettoie la mémoire et ferme tous les fichiers MED ouverts (de même que tous les fichiers HDF). Attention donc à l'emploi de cette routine qui doit utilisée en cas de force majeure dans un contexte d'E/S parallèles.
